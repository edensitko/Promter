# E-Commerce Store Example

> Generated using the **ecommerce-modern** preset.

## Overview

This example demonstrates a modern e-commerce storefront generated using the `ecommerce-modern` preset, which combines `modern-ui`, `ecommerce-layout`, `hover-effects`, and `animated-cards` prompts.

## Generated Sections

### 1. Navigation Header
A sticky header with logo, category mega-menu, search bar with autocomplete, user account dropdown, and cart icon with animated badge counter. Transparent on hero, becomes solid white with shadow on scroll.

### 2. Hero Banner
A full-width promotional banner with large product imagery, headline text, and "Shop Now" CTA. Supports an auto-rotating carousel for multiple promotions. Smooth crossfade transitions between slides.

### 3. Category Grid
A visual category browser with large image cards (Men, Women, Accessories, Sale). Each card features a zoom hover effect on the background image and a sliding text overlay. Links to filtered product listing pages.

### 4. Featured Products Section
A curated grid of 8 featured products in a responsive 4-column layout. Product cards include image, title, price, rating, and quick-add button. Hover triggers image swap (front/back view) with a fade transition and reveals a quick-view button overlay.

### 5. Product Listing Page
The main shop page with a sidebar filter panel (category, price range, size, color, rating) and a responsive product grid. Supports grid/list view toggle. Products animate in with staggered fade-up on filter changes.

### 6. Product Quick View Modal
A modal triggered from product cards showing larger images, size selector, quantity picker, and add-to-cart button. Image gallery with thumbnail navigation. Smooth scale-and-fade entrance animation.

### 7. Promotional Banner Strip
A mid-page promotional strip with countdown timer, discount messaging, and CTA. Animated gradient background for attention.

### 8. Cart Sidebar
A slide-in cart panel from the right side. Displays cart items with quantity adjustment, remove buttons, subtotal, and checkout CTA. Items animate in/out of the cart list with smooth transitions.

### 9. Footer
Multi-column footer with shop categories, customer service links, payment method icons, social links, and newsletter signup. Trust badges and security seals.

## Technologies Used

- **React** with Next.js App Router
- **Tailwind CSS** for styling
- **Framer Motion** for card animations and hover effects
- **TypeScript** for type safety

## Component Count

Approximately 25–30 components including product cards, filter controls, cart components, and layout elements.
