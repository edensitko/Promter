# Landing Page Example

> Generated using the **startup-landing** preset.

## Overview

This example demonstrates a startup landing page generated using the `startup-landing` preset, which combines `modern-ui`, `landing-layout`, `scroll-animations`, and `bento-grid-layout` prompts.

## Generated Sections

### 1. Hero Section
A full-viewport hero with a bold headline, subheadline, and dual CTA buttons. Background features a subtle gradient mesh with floating geometric shapes. Text enters with a staggered fade-up animation on page load.

### 2. Logo Bar
A horizontal strip of partner/client logos in muted grayscale. Logos subtly colorize on hover. Infinite horizontal scroll animation for visual interest.

### 3. Bento Feature Grid
A 4-column asymmetric bento grid showcasing product features. Large feature cards span 2 columns, smaller cards occupy single cells. Each card contains an icon, title, and short description. Cards reveal with staggered scroll animations as they enter the viewport.

### 4. Product Screenshot Section
A large product screenshot or mockup centered on screen with a subtle parallax tilt effect. Flanked by floating feature callouts that animate in from the sides on scroll.

### 5. Social Proof / Testimonials
A carousel of customer testimonials with avatar, name, role, and quote. Cards feature a subtle glass morphism effect. Auto-advances with manual navigation controls.

### 6. Pricing Section
Three-tier pricing table (Free, Pro, Enterprise) with a highlighted recommended plan. Clean card layout with feature checklists. Toggle between monthly and annual billing with animated price transition.

### 7. FAQ Section
An accordion-style FAQ with smooth expand/collapse animations. Each answer reveals with a slide-down transition.

### 8. CTA Section
A prominent call-to-action banner with gradient background. Bold headline with email signup or "Get Started" button. Subtle background animation for visual interest.

### 9. Footer
Multi-column footer with navigation links, social icons, newsletter signup, and legal links. Clean, organized layout matching the modern UI style.

## Technologies Used

- **React** with Next.js App Router
- **Tailwind CSS** for utility-first styling
- **Framer Motion** for scroll animations and entrance effects
- **TypeScript** for type safety

## Component Count

Approximately 15–20 components including layout wrappers, section components, and reusable UI elements.
